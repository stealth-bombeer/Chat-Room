
# Chat-Room

A One-to-One/Group chat application  


## Features

- User can join a chat using Room-Pin
- All the people in the same room can view and send message
- Live previews



## Deployment

To host this project run

```bash
server-side rendering

  cd server
  npm install
  nodemon app
```

```bash
client-side rendering 

cd client 
npm install
npm run start
```
